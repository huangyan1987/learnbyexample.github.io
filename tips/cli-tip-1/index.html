CLI tip: 1

2021-11-18

Want to remove metadata (DateTime, Model, Orientation, ShutterSpeedValue, etc) from your images? Here's an example using mogrify for all files ending with .jpg in the current directory.

$ mogrify -strip *.jpg

Note that this will replace your original images and output size may vary because of recompression. See the below resource links for more details and alternate tools.
